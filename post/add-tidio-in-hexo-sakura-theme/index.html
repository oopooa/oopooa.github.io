Hexo(Sakura) 添加在线聊天 Tidio

Sakura 主题安装免费在线聊天 Tidio

2020-04-26
阅读时长: 1 分钟

前言

一直想给博客装款在线聊天软件，综合对比下来后，选择了 Tidio ，其界面优雅简洁，支持多渠道回复，网页、Windows、安卓、IOS，关键是免费功能对普通用户完全够用，所以理论上来说是完全免费的！(白嫖使我快乐)

正文

安装起来还是比较简单的，但因为是国外的网站，访问起来会有一些慢，需要点耐心。

注册账号

访问 Tidio 注册账号，一个邮箱就能搞定，非常方便。

自定义Tidio

进入账户界面后，可以根据自己的喜好定制，可选颜色，语言，布局，侧边栏等等。

找到账户面板的 Public Key

安装

在主题配置文件 _config.yml 添加如下代码

1
2
3
4

# Tidio online chat
tidio:
  enable: true
  key: # 替换为你的Public Key

在 themes/Sakura/layout/_partial/footer.ejs 中添加如下代码

1
2
3
4

<!-- 在线通讯Tidio -->
<% if (theme.tidio.enable){ %>
<script src="//code.tidio.co/<%- theme.tidio.key %>.js"></script>
<% } %>

到此 Tidio 的安装就完成了，Hexo 三连后刷新页面，在右下角就能看到效果。

1

hexo clean && hexo g && hexo s
